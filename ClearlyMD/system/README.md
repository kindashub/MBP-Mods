# ClearlyMD — `system/`

Scripts and docs for the ClearlyMD mod. **Clickable apps** (**ClearlyMD.app**, **ClearlyEdit.app**) live in the parent **`ClearlyMD/`** folder.

| File | Role |
|------|------|
| **ClearlyMD-SetupGuide.md** | Full instructions — start here. |
| **setup-clearlymd.sh** | One-shot install (launcher, editor, Dock, `duti`). |
| **install-clearlymd.sh** | Download **ClearlyMD.app** from Releases. |
| **build-clearlyedit-app.sh** | Build **ClearlyEdit.app** in parent folder. |
| **clearlyedit-new-md.sh** | Source for **`clearlyedit`** (copied by setup). |
