# MBP-Mods

**Repository:** [github.com/kindashub/MBP-Mods](https://github.com/kindashub/MBP-Mods)

macOS reinstall stash: **modified apps, scripts, and blueprints** collected in one place. Use this when you wipe or migrate a machine—clone the repo (or copy the folder from backup), then follow each mod’s guide.

## Layout

Each **mod** has its own top-level folder (self-contained).

| Folder | Contents |
|--------|----------|
| **TextMD** | TextEdit + `~/TextMD` workflow — Dock app, launcher script, `TextMD-SetupGuide.md`. |
| **ClearlyMD** | ClearlyMD editor + ClearlyEdit Dock helper — same `~/TextMD` naming; `ClearlyMD-SetupGuide.md`, `install-clearlymd.sh`, `setup-clearlymd.sh`. |

Add new mods as new folders here.
