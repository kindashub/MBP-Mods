- **2026-04-08 — Dock “nothing ever changes” (real bug):** `kindasmd` used `open -a`, which **reuses any running `com.kindasmd.editor`** (e.g. old **Xcode DerivedData**). The installed **`KindasMD/KindasMDEditor.app` never launched** — all builds looked broken. Fix: **`$HOME` / `getpwuid`**, reject `…/Library/Containers/…`, re-resolve default path every **`ensureFolderAndRefresh()`**; **ContentView** calls refresh on appear + strip toggle + mode change; **Picker** uses **Int** tags. Build **7**.
